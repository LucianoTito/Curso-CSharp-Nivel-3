# 💻 Portfolio Profesional - Luciano Tito Cedrón

Bienvenido al repositorio de mi portfolio personal. Este proyecto nació originalmente como una maqueta estática (HTML/CSS/JS) y fue **refactorizado y migrado hacia una arquitectura ASP.NET WebForms**, reflejando mi transición y especialización actual hacia el Desarrollo Backend.

## 🚀 Sobre el Proyecto

El objetivo de esta migración fue implementar un entorno de servidor estructurado en **.NET Framework**, aplicando buenas prácticas de desarrollo y preparándolo para futuras integraciones lógicas y bases de datos.

### 🔧 Características Técnicas Destacadas:
* **Arquitectura de Plantillas:** Uso de `Site.Master` para modularizar la interfaz de usuario (Navegación y Footer), promoviendo la reutilización de código y el mantenimiento eficiente.
* **Enrutamiento del Lado del Servidor:** Navegación gestionada a través de páginas de contenido independientes (`.aspx`), separando las vistas y la lógica.
* **Internacionalización (i18n):** Sistema de cambio de idioma dinámico (Español/Inglés) consumiendo archivos `.json` mediante fetch API en JavaScript.
* **UI/UX Interactiva:** Implementación de "Dark Mode" con persistencia de datos usando `localStorage`.
* **Validación de Formularios:** Control de campos obligatorios en el Frontend mediante JavaScript antes del envío al servidor.

## 🛠️ Stack Tecnológico

* **Backend & Framework:** C#, ASP.NET WebForms (.NET Framework)
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Control de Versiones:** Git & GitHub
* **IDE:** Visual Studio

## 📂 Estructura del Proyecto

* `Site.Master`: Molde principal de la aplicación.
* `Default.aspx`: Página de inicio y presentación de proyectos.
* `Contacto.aspx`: Formulario de contacto preparado para procesamiento backend.
* `/src`: Directorio de recursos estáticos (CSS, JS, iconos e imágenes).
* `/languages`: Archivos JSON para la traducción dinámica.
